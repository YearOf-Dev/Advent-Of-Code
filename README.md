## Languages

Alongside the actual challenges, I am testing how long the implementation takes in each language. I do this is three parts:
- The time it takes to run the part 1 function
- The time it takes to run the part 2 function
- The time it takes to read the input, and run both functions

All of this is measured either by dedicated performance metrics built into the language, or by taking timestamps of start and end and calculating the difference. This is recorded in Nanoseconds. Each challenge has a winner and a loser for each of these three parts.
